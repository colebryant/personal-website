MixMate

Nashville Software School Front-End Capstone

As someone who enjoys making cocktails but finds managing inventory and tracking recipes difficult, I decided to build MixMate, a mobile-first web application. The app enables users to create and search cocktail recipes, manage inventory, and determine which cocktails can be made with the ingredients the user has in stock, among other features. MixMate was built using React.js with React-Select for field selection, Auth0 for login and registration and Bulma for styling.

Nomads

Nashville Software School Front-End Group Project

Nomads is a simple front-end social media app with a theme centered around van life travelers, and was built as part of a group project at Nashville Software School. Users can view articles, check out upcoming events, organize their tasks, message other users and add friends. The app was built using vanilla JavaScript with Bootstrap for styling and Grunt and Browserify for task automation. I worked on the events feature in addition to the ERD.

Bangazon-Workforce

Nashville Software School Back-End Group Project

Part of a series of back-end sprints at Nashville Software School aimed at building applications for a mock company, Bangazon-Workforce is an employee management platform which enables users to easily interact with employee database resources. Our group built this application using C# /.NET MVC and leveraged Razor Pages for the User Interface. I was responsible for building the controllers (and associated views, models, and view-models) for the computer and department resources.
